Review: ES6 Classes

  • Classes are a template for creating objects.
  • A class declaration cannot be hoisted.
  • A constructor is a special method within a class used to initialize objects. It is called automatically when a new object is created. The contextual "this" refers to the current object instance.

Using Express Routing

  • Within Express, routing refers to the process of determining how an application responds to a client request to a particular URL and HTTP method.
  • A route path in Express defines the URL pattern to match against incoming requests.
  • A route method in Express specifies the HTTP method that the route will respond to.
  • It is appropriate to add "next" as a parameter to a route handler when you want to pass control to the next middleware or route handler function.

Express Routing

  • An Express Router is a mini express application that provides a modular way to handle routes.
  • We initialize an "express.Router()" in an express server by creating a new instance of the router using the "express.Router()" function.
  • Route middleware in Express is used to perform operations on the request and response objects or to add additional functionality before passing control to the next middleware or route handler.
