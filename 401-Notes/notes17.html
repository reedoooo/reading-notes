Class 17: S3 and Lambda

May 17, 2023

AWS S3

  • Amazon S3 is a scalable object storage service for storing and retrieving data.
  • Use cases for S3 include backup and restore, content distribution, and static website hosting.
  • Benefits of S3 include scalability, reliability, security, and cost optimization.

AWS Lambda

  • AWS Lambda lets you run your code without provisioning or managing servers.
  • Use cases for Lambda include real-time file processing, stream processing, and ETL operations.
  • "Serverless" is a way of running applications without having to manage any servers.

CDN

  • A CDN delivers content to users based on their geographic location.
  • A CDN improves website load times, decreases bandwidth costs, and increases content availability.
