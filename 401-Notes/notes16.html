AWS EC2

  • An EC2 instance is a virtual server for running applications on the AWS infrastructure.
  • EC2 can be used for hosting web apps or websites and for data processing tasks.
  • ECS is preferred over Heroku, Digital Ocean, or Render.com due to its tight integration with other AWS services.
  • EC2 can be found in the AWS Console under the "Compute" section.
  • T2 Micro is smaller and more economical, suitable for low-traffic applications, while XL offers more computing power and memory.
  • A "Compute Cycle" is the work a computer can do in one "tick" of its internal clock.

Elastic Beanstalk

  • Elastic Beanstalk is a fully managed service by AWS for deploying and running applications.
  • Elastic Beanstalk uses EC2 instances to run the deployed applications.
  • Benefits of Elastic Beanstalk include automatic scaling, ease of starting, and full control over the underlying resources.
