React Quick Start

  • Building blocks of a React app: The main building blocks of a React app are components, which are JavaScript classes or functions that optionally accept inputs (props) and return a React element.
  • Difference between an HTML element and a React component: An HTML element represents a webpage element like a button or a paragraph, while a React component is a JavaScript class or function that accepts inputs and returns a React element.
  • JSX and its usage: JSX is a syntax extension for JavaScript that allows you to write HTML-like code in your JavaScript files. It's used in React to define the structure and appearance of a component's UI.
  • JavaScript expressions in JSX: JavaScript expressions can be embedded in JSX by wrapping them in curly braces.
  • React's features for iteration or conditional logic: React doesn't have special syntax for loops or conditions. Instead, it leverages JavaScript's native conditional and looping constructs.
  • React's response to user inputs: React responds to user inputs by using a system of "events", which are triggered on user interactions and can be handled with functions.
  • Indication of a React component managing data with a Hook: The use of a Hook in a React component is indicated by the prefix "use" in a function.
  • Sharing data between two React components: Two React components can share data by passing it via props, or by using state and context, or a state management library like Redux.

Render and Commit

  • Three steps of refreshing a React UI: The three steps are "render", "commit", and "update".
  • Triggering updates to a component after the initial render: Updates can be triggered by state changes, prop changes, or forced updates.
  • React's recreation of DOM nodes on every rerender: React doesn't recreate DOM nodes on every rerender. It uses a "diffing" algorithm to identify changes and only updates those.
  • Steps after React has updated the DOM: After the DOM has been updated, the browser's rendering engine must paint the changes to the screen.
