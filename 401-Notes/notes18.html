AWS API Gateway

  • Amazon API Gateway is a fully managed service for creating and managing APIs.
  • API Gateway is crucial in the Serverless ecosystem as it handles concurrent API calls and integrates with other AWS services.
  • Benefits of API Gateway include high performance, cost savings, and flexibility.
  • You can choose from RESTful APIs and WebSocket APIs.

AWS DynamoDB & Dynamoose

  • DynamoDB is a high-performance, scalable key-value and document database.
  • DynamoDB is recommended over MongoDB for applications requiring a fully managed, serverless noSQL database.
  • Explaining DynamoDB to a friend: It's like a huge, smart and very fast filing cabinet for data.
  • Dynamoose is a modeling tool for DynamoDB, offering features like automatic request retries and support for all DynamoDB data types.
