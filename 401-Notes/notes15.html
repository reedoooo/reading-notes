Tree Analogy

  • Trees in computer science are like real-world trees turned upside down. The root of the tree is at the top, branches extend from the root, and those branches end in nodes, which can be thought of like leaves.

Vocabulary/Definition List

  • Tree: A hierarchical structure of linked nodes.
  • Root: The top node in a tree.
  • Child: A node directly connected to another node when moving away from the Root.
  • Parent: The converse notion of a child.
  • Siblings: Nodes with the same parent.
  • Leaf: A node with no children.
