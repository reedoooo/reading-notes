Learn CSS Flexbox

  • Flexbox is a one dimensional layout method for organizing rows and columns. This means items 'flex' to fill space or shrink into place.
  • The cross axis in the flex box model runs perpendiclar to the layout of the box items while the main axis runs horizontal.
  • A disadvantage to flexbox is that it'll generally increase the load time of websites because of it's dynamic capabilities.

CSS Layout Flexbox

  • Advantages of flexbox: Positioning child elements is easier. The order of elements on the page can be more easily changed. Flexbox is responsive and thus works better for mobile. Flex containers' margins don't collapse with the margins of it's content.
  • In the future I'm going to be building websites or webpages for companies that must be easily accessible and fully function. Learning to use flexbox is essential in creating the best webpages CSS can offer.
