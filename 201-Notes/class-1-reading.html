Introduction to HTML

  1. HTML Attributes: An html attribute as an identifier put inside an element to allow it to be specifically target for styling by javascript.
  2. Anatomy of an HTML element: An HTML elements begins with a less than sign and ends with a greater than sign and it has a word inbetween which allows it to give the basic form of a website. It can be given an ID or Class. It must be closed with a closing tag.
  3. Article tags vs. section tags: Section tags are gnerally used to break things up into subcomponents wheras an article somewhat of a primary component of a website.
  4. Elements of a typical website: It will have the html tag, as well as the head and main tag. It will sometimes have a header and always has a body tag and sometimes a footer. Websites also generally use div tags and tags for headers.
  5. Metadata influence on search engines: It helps search engines understand what your website is about and how it should be indexed.
  6. How the meta tag is used: It will have a name part for a descript and a content part for a longer description. It goes inside the head tag.
