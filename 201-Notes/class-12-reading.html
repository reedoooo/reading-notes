Javascript Canvas

  • The <*canvas> tag allows a developer to create dynamic, scriptable graphics on a web page.
  • The closing <*/canvas> tag is important because it indicates the end of the canvas element and any content within it.
  • The getContext() method is used to obtain a rendering context for the canvas, which can be used to draw and manipulate graphics on the canvas.

Chart.js Documentation

  • Chart.js is a JavaScript library that allows developers to easily create charts and graphs on a web page.
  • Chart types that can be created using Chart.js include: line charts, bar charts, and pie charts.

Animated Charts

  • Advantages of displaying data via a chart over a table include: improved visual representation of data, easier to compare data points, and improved readability.
  • Chart.js can aid previously created applications visually by allowing for the creation of interactive and dynamic charts that can make the data more engaging and easier to understand.
