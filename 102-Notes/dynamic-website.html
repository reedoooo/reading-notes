Dynamic Web Pages Using Javascript

Variables

  • There are three types of variables. Var is the original way of setting a simple variable. Let creates a variable of which the value might be changed later. Const creates a variable with a set value.
  • Javascript variables are identified with unique named called identifiers.
  • The (=) is used to assign value to a variable. The (==) sign is set a varible equal to something.
  • Variables can be set as "text"/'text' called strings and numbers with no quotes 10.
  • Good practice is declaring variables at the beginning of a script.
  • If a variable is not assigned a value (i.e. let reedVogt;) it is automatically set to undefined.
  • A line that begins with a string stays a string but if it begins with a number you can add a string

Beginner to Advanced Concepts

  • Beginner concepts include: Conditional statements, loops, functions and events. Other concepts include objects, asynchronous javascript, and APIs
  • Intermediate concepts include: Client-side javascript frameworks, data structures, equalty comparisons, closurs and enumerability and ownership of properties.
  • Advanced concepts include: Inheritence and the prototype chain, typed arrays, memory management, and concurrency models and event loops.

Using Prompt() and Confirm()

  • The prompt() function will create a pop-up window displaying a form for users to fill in as well as a cancel and ok button.
  • When the user presses OK the value in the prompt() function will be returned.
  • Then the document.write("", ) method will update the html with the resulting value.
