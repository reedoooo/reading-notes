Operators and Loops Notes

Comparison Operators and Assignment Operators

  • Comparison Operators
    • Comparison operators allow you to compare two values and determine if they are equal, greater than, or less than each other.
    • The value returned will be boolean, indicating true or false
    • Comparison operators can be used with any data type in JavaScript, but there are some specific rules for comparing strings and objects
  • Assignment Operators
    • Assignment operators are used to assign a value to a variable.
    • Some examples of assignments operators are: ("="), ("+="), and ("-=").
    • The first assignment operator above is simple whereas the following operator is compound.

For Statements and While Statements Loops

  • For Statement Loops
    • For statement loops are used when you want to repeat a block of code a specific number of times.
    • The syntax for a for loop is:

      for (initialization; condition; increment) {
      code block to be executed
      }
    • The initialization is executed before the loop starts, the condition is checked at the beginning of each iteration, and the increment is executed at the end of each iteration. If the condition is true, the code block is executed. If the condition is false, the loop ends.
    • The increment or decrement statement is executed at the end of each iteration and is used to update the loop counter
  • While Statement Loops
    • While statement loops are used when you want to repeat a block of code an unknown number of times. The while loop executes a block of code as long as a condition is true.
    • The syntax for a while loop is:

      while (condition) {
      code block to be executed
      }
    • The condition is checked at the beginning of each iteration. If the condition is true, the code block is executed. If the condition is false, the loop ends.
    • While loops are often used when the number of iterations is not known in advance
    • The break and continue statements can be used to interrupt or skip iterations in a while loop
