Javascript Function Notes

MDN Control Flow

  • Control flow is the order in which the computer executes code.
  • Conditional statements such as if and else are used to control the direction in which code is executed.
  • Loops, functions and events are other forms of control flow.
  • The parentheses in a function may include parameters such as strings, numbers or other forms of data
  • Arguments are the values received by the function when it is invoked

Functions

  • A function is a block of code designed to execute certain actions.
  • Function are executed when called on like this: function();
  • A function will be invoked when: An event occurs, it's called, or automatically.
  • When JS reaches a return statement it will stop executing.
  • A function's return value is returned back to the caller.

Operators

  • An Assignment Operator (=) assigns a value to a variable.
  • The Addition Operator (+) adds numbers.
  • The Subtraction Operator (-) subtracts numbers.
  • The Multiplication Operator (*) multiplies numbers.
