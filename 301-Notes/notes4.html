React Docs - Forms

  • A controlled component is a form element whose value is controlled by React. Instead of letting the DOM handle the form data, React handles the form data and updates the component state on every change.
  • We should update the state with users' responses as soon as they enter them. This is because in controlled components, React maintains the single source of truth for the input field, making it easier to maintain and manage the component's state.
  • We can use the event.target.value property to access what the user is entering in the input field.

The Conditional (Ternary) Operator Explained

  • The conditional (ternary) operator is a concise way to write conditional statements in JavaScript.
  • We would use a ternary operator to write a more concise and readable code in place of an if...else statement.
  • x === y ? console.log(true) : console.log(false);
