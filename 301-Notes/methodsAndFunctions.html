Arrays

  • Array.forEach()

    The forEach() method executes a provided function once for each array element.

    const addValues = (arr, value) => { arr.push(value); }; const addNumbers = (num, arr, times, callback) => { const iterations = new Array(times).fill(null); iterations.forEach(() => { callback(arr, num); }); return arr; };

    Example: A function named addValues that takes in an array and a value and pushes the value into the array. This function does not need a return statement..

  • Array.map()

    The map() method creates a new array populated with the results of calling a provided function on every element in the calling array.

    const charCode = (arr) => { const newArray = arr.map((num) => num.charCodeAt(0)); return newArray; };

    Example: A function named charCode that, given an array of letters as an input, uses map to return a new array where each element is the result of the `charCodeAt` method on the original array element.

    const toLastNames = (people) => { let firstLastList = people.map((person) => { let firstLast = person.firstName + ' ' + person.lastName; return firstLast; }); return firstLastList; };

    Example: A function that iterates over an array of people objects and creates a new list of each person's full name using the array method 'map'.

  • Array.sort()

    The sort() method sorts the elements of an array in place and returns the reference to the same array, now sorted. The default sort order is ascending, built upon converting the elements into strings, then comparing their sequences of UTF-16 code units values.

    const alphabetizeBetter = (arr) => { return arr.sort((a, b) => a.toLowerCase().localeCompare(b.toLowerCase())); };

    Example: A function named alphabetizeBetter that takes in an array of strings and returns the same array, with the strings sorted alphabetically. Capitalization should not change the sort order of two strings.

  • Array.filter()

    The filter() method creates a shallow copy of a portion of a given array, filtered down to just the elements from the given array that pass the test implemented by the provided function.

    const notInFirstArray = (forbiddenValues, arr) => { return arr.filter((num) => { return !forbiddenValues.includes(num); }); };

    Example: A a function named notInFirstArray that, given two arrays as input, uses filter to return an array of all the elements in the second array that are not included in the first array.

  • Array.reduce()

    The reduce() method executes a user-supplied "reducer" callback function on each element of the array, in order, passing in the return value from the calculation on the preceding element. The final result of running the reducer across all elements of the array is a single value.

    const reversedString = (str) => { const reverseName = str.split('').reduce((previous, current) => { return current + previous; }, ''); return reverseName; };

    Example: A function named reversedString that takes in a string and returns a string with the letters in reverse order.

  • Array.pop()

    The pop() method removes the last element from an array and returns that element. This method changes the length of the array.

    const fruits = ['apple', 'banana', 'pear', 'kiwi']; const lastFruit = fruits.pop(); // kiwi console.log(fruits); // ['apple', 'banana', 'pear']

    Example: An array of fruits is created and the last element is removed using the pop() method, which is then stored in the variable 'lastFruit'. The output will display the array without the last element.

  • Array.fill()

    The fill() method changes all elements in an array to a static value, from a start index (default 0) to an end index (default array.length). It returns the modified array.

    const numbers = [1, 2, 3, 4, 5]; numbers.fill(0, 2, 4); // [1, 2, 0, 0, 5]

    Example: An array of numbers is created and the fill() method is used to change the elements at index 2 and 3 to 0. The output will display the modified array.

  • Array.toLocaleString()

    The toLocaleString() method returns a string representing the elements of the array. The elements are converted to Strings using their toLocaleString methods and these Strings are separated by a locale-specific String (such as a comma ",").

    const numbers = [1000, 2000, 3000]; const formattedNumbers = numbers.toLocaleString('en-US', { style: 'currency', currency: 'USD' }); // $1,000.00,$2,000.00,$3,000.00

    Example: An array of numbers is created and the toLocaleString() method is used to format the numbers as currency in US dollars. The output will display the formatted numbers as a string.

  • Array.toString()

    The toString() method returns a string representing the specified array and its elements.

    const arr = [1, 2, 3, 4, 5]; const arrString = arr.toString(); // '1,2,3,4,5'

  • Array.shift()

    The shift() method removes the first element from an array and returns that removed element. This method changes the length of the array.

    const arr = [1, 2, 3, 4, 5]; const first = arr.shift(); // 1 console.log(arr); // [2, 3, 4, 5]

  • Array.includes()

    The includes() method determines whether an array includes a certain value among its entries, returning true or false as appropriate.

    const arr = [1, 2, 3, 4, 5]; const includesThree = arr.includes(3); // true const includesSix = arr.includes(6); // false

  • Array.join()

    The join() method creates and returns a new string by concatenating all of the elements in an array (or an array-like object), separated by commas or a specified separator string. If the array has only one item, then that item will be returned without using the separator.

    const arr = ['apple', 'banana', 'cherry']; const joined = arr.join(', '); // 'apple, banana, cherry'

  • Array.splice()

    The splice() method changes the contents of an array by removing or replacing existing elements and/or adding new elements in place. To access part of an array without modifying it, see slice().

    const arr = [1, 2, 3, 4, 5]; arr.splice(2, 1); // removes 1 element at index 2 console.log(arr); // [1, 2, 4, 5] arr.splice(2, 0, 'a', 'b'); // adds 'a' and 'b' at index 2 console.log(arr); // [1, 2, 'a', 'b', 4, 5]

  • Array.slice()

    The slice() method returns a shallow copy of a portion of an array into a new array object selected from start to end (end not included) where start and end represent the index of items in that array. The original array will not be modified.

    const arr = ['apple', 'banana', 'cherry', 'date', 'elderberry']; const newArr = arr.slice(1, 4); console.log(newArr); // Output: ['banana', 'cherry', 'date']

    Example: Using slice() to create a new array containing a portion of the original array.

  • Array.reverse()

    The reverse() method reverses an array in place and returns the reference to the same array, the first array element now becoming the last, and the last array element becoming the first. In other words, elements order in the array will be turned towards the direction opposite to that previously stated.

    const arr = [1, 2, 3, 4, 5]; arr.reverse(); console.log(arr); // Output: [5, 4, 3, 2, 1]

    Example: Using reverse() to reverse the order of an array.

Strings

  • String.charAt()

    The charAt() method of a String instance returns a new string consisting of the single UTF-16 code unit located at the specified offset into the string.

  • String.localeCompare()

    The localeCompare() method returns a number indicating whether a reference string comes before, or after, or is the same as the given string in sort order. In implementations with Intl.Collator API support, this method simply calls Intl.Collator.

  • String.substring()

    The substring() method returns the part of the string from the start index up to and excluding the end index, or to the end of the string if no end index is supplied.

  • String.split()

    The split() method takes a pattern and divides a String into an ordered list of substrings by searching for the pattern, puts these substrings into an array, and returns the array.
