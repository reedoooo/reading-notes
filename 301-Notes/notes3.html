React Docs - lists and keys

  • The .map() method returns a new array with the results of calling a provided function on every element in the original array.
  • We can use the .map() method to loop through the array and return a new array of JSX elements that represent each value.
  • Each list item needs a unique "key" prop
  • The "key" prop is used by React to keep track of which items have changed, been added, or been removed. It helps optimize the rendering process and improve performance by minimizing unnecessary re-renders.

The Spread Operator

  • The spread operator is a syntax for expanding an iterable (e.g. an array or an object) into a list of arguments or elements.
  • The spread operator can:
    • Combine arrays or objects
    • Copy an array or object
    • Add elements to an array
    • Pass arguments to a function
  • Using the Spread Operator to Combine two Arrays 
                                
                            const array1 = [1, 2, 3];
                            const array2 = [4, 5, 6];
                            const combinedArray = [...array1, ...array2];
                            console.log(combinedArray);
  • Using the Spread Operator to Add a New Item to an Array 
                                
                            const originalArray = [1, 2, 3];
                            const newArray = [...originalArray, 4];
                            console.log(newArray);
  • Using the Spread Operator to Combine two Objects into One 
                                
                            const obj1 = { a: 1, b: 2 };
                            const obj2 = { c: 3, d: 4 };
                            const combinedObject = { ...obj1, ...obj2 };

How to Pass Functions between Components

  • The developer creates a function in the parent component and passes it as a prop to the child component.
  • The increment function is a callback function that increases the count in the state of the parent component by one. It is passed down to the child component as a prop, which allows the child component to update the count in the parent component.
  • You can pass a method from a parent component into a child component by defining the method in the parent component and passing it as a prop to the child component.
  • The child component can invoke a method that was passed to it from a parent component by calling the method using the props object and passing any necessary arguments. For example: this.props.methodName(argument1, argument2)
