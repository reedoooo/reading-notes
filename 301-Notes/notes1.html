Component-Based Architecture:

  • Definition of a component: A component is a self-contained unit of a user interface.
  • Characteristics of a component: Reusable, composable, and modular.
  • Advantages of using component-based architecture: Improves code reusability, maintainability, and structure.

Props and How to Use it in React:

  • Definition of "props": Short for "properties".
  • Use of props in React: Used to pass data from a parent component to a child component.
  • Flow of props: Passed from parent to child component as immutable values.
