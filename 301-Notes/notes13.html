CRUD Basics

  • To update a record through an API, the HTTP method commonly used is PUT or PATCH.
  • The HTTP methods used for the CRUD operations are:
    • Create: POST
    • Read: GET
    • Update: PUT or PATCH
    • Delete: DELETE
  • All REST methods require a resource identifier (ID) parameter, except for the CREATE method which generates a new ID automatically.

Speed Coding: Building a CRUD API

  • REST is an architectural style for building web services, while CRUD is a set of basic operations performed on data in a database or API.
  • The steps for creating a RESTful API can be summarized as:
    1. Define the resources and the endpoints (URLs).
    2. Decide on the HTTP methods to be used for each endpoint.
    3. Design the request and response data structures.
    4. Implement the API endpoints and the corresponding CRUD operations.
    5. Test and document the API.
